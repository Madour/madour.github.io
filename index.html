My projects

Park Conquest

Park Conquest is a tactical RPG video game I am creating from scratch.

I started this project back when I was in highschool as one of my biggest long term project.

NasNas

NasNas is a C++ game framework i am currently developing. It is built on top of SFML.

The framework makes game development less tedious by taking care of all the repetitive tasks.

It provides a bunch of features like resource loading and management, a scene and layers system, game cameras, and more.

You can find it on GitHub.

pyNasNas

pyNasNas is the python version of NasNas. It is a bit ahead in development.

It has the same API as its sibling, and follows Python code style guide.

It is be useful for fast game prototyping before switching to C++ when you need performance.

You can find it on GitHub.
