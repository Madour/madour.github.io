My projects

NasNas

NasNas is a beginner friendly C++ game framework I am currently developing.

It provides a bunch of features out of the box like resource loading and management, a scene and layers system, an data driven entity component system, tile mapping, transitions, sprite batching, tweening and more.

You can find it on GitHub.

LDtk Loader

LDtk Loader is a fast and light loader for projects created with the awesome Level Designer toolkit.

It has a simple API and can work with any rendering backend. Examples using SDL or SFML for rendering the levels are available.

You can find it on GitHub.

GameJams Games

When I have some free time, I like to participate in game jams with specific rules and constraints.

You can find some of my games on Itch.io.
